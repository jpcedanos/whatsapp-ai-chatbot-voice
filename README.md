# WhatsApp AI Chatbot with Voice Cloning

An advanced AI chatbot integrated with WhatsApp, capable of generating human-like voice responses using text-to-speech and voice cloning technologies.

## Features
- Seamless WhatsApp integration
- High-quality Text-to-Speech (TTS)
- Custom voice cloning
- Persistent conversation memory

## Tech Stack
- Python
- Gemini API (LLM)
